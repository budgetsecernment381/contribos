## 🚀 What is contribos?

contribos is a Windows app that helps you find open source issues, draft fixes with AI, review the code changes, and prepare pull requests.

It is made for people who want to contribute to GitHub projects without spending a lot of time on setup or manual search.

You can use it to:

- Find issues that match your skills
- Generate a code fix with LLM agents
- Review the diff before you submit anything
- Connect your own AI provider key
- Work with the A2A protocol for agent-based workflows

## 🛠️ Install on Windows

1. Open the releases page from the link above
2. Find the latest version at the top of the list
3. Under Assets, download the Windows installer
4. When the file finishes downloading, open it
5. If Windows asks for permission, choose Yes
6. Follow the setup steps on screen
7. Finish the install and open contribos from the Start menu or desktop shortcut

If the file is a `.zip`, open it after the download finishes, then move the app to a folder like `Downloads` or `Desktop` before you run it

## 🔑 AI provider setup

contribos supports many LLM providers, so you can choose the one you already use.

Common setup fields may include:

- Provider name
- API key
- Model name
- Base URL
- Project or workspace settings

If you use BYOK, you bring your own key and keep control of your account use. This can help if you already have access to a provider and want to keep your work in one place

## 🧭 How contribos works

A simple workflow looks like this:

1. Search for an open source issue
2. Open the issue in contribos
3. Let an agent suggest a fix
4. Review the diff
5. Make any changes you want
6. Submit a pull request to GitHub

This keeps the process in one place and helps you move from issue to PR with less manual work
